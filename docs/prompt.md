### 2026-06-30
사다리 타기 도구 개선 사항
1. 모든 도구는 SPA로 구현해야 함
- 사용자가 쉽게 사용하기 편해야 함 (쉬운 UX, 사용성이 무엇보다 중요)
- 이 지침을 하네스에 등록해줘
2. 도움말은 기본으로 접혀있어야 함(닫기).
- 필요시 사용자가 열면 됨
3. 플레이어 이름에 자동으로 추천하는 모드가 있어야 함
- 이 모드가 활성화되면 이모지의 과일을 랜덤으로 섞어서 플레이어 이름으로 추천되어야 함
4. 결과 자동으로 추천하는 모드가 있어야 함
- 1명만 당첨되는 모드를 클릭하면 당첨, 나머지는 꽝
- 순위로 당첨되는 모드를 클릭하면 이모지 숫자를 랜덤으로 추천(1부터 시작)
  - 10 이상이면 이모지를 2개 사용
5. 결과 복사는 결과 다운로드로 변경하여 이미지가 다운로드 될 수 있도록 해야 함
6. 사다리의 마지막 가로 선 모양이 바닥과 붙어 있음
- 시작 처럼 마지막에 가로선이 연결되어 있으면 안됨 
- 가로선의 시작과 마지막은 동일한 간격으로 시작하고 종료되어야 함
7. 참가자를 클릭할 경우 결과가 시뮬레이션 선이 생겼다가 없어짐
- 계속 보이게 두어야 함
8. 소리 기능을 확인해줘
9. 그 밖에 개선해야 할 포인트를 찾고 제안해줘
10. 에러를 재현하고 TDD로 구현
